# H5 使用说明

## 地址

启动后端后打开：

```text
http://127.0.0.1:8010/h5
```

## 当前流程

1. 输入专家问题。
2. 点击“解析预览”。
3. 页面展示候选规则和政策依据。
4. 默认选中第一条候选规则，也可以手动切换候选规则。
5. 设置开始日期和结束日期。
6. 点击“确认并生成 SQL”。
7. 页面展示受控 SQL、参数和安全校验结果。
8. 点击“模拟执行”。
9. 页面展示汇总指标、机构排名、月度趋势和疑点明细样例。
10. 点击“查看任务”可重新读取任务结果。
11. 点击“导出 CSV”可导出疑点明细样例。

## 示例问题

```text
查询急诊监护费和急诊诊查费是否存在重复收费
```

可返回候选规则：

```text
急诊监护费|急诊诊查费
001103000010000|001102000030000
项目A和项目B同时收费
```

## 注意

当前 H5 跑 MVP 演示闭环：

```text
专家问题 → 规则/RAG 预览 → 专家确认 → 生成受控 SQL → 模拟执行结果
```

当前不会执行 SQL，也不会连接真实医保数据库。模拟执行结果用于先验证产品流程、页面交互和接口结构。

## 快速查询与依据链

页面左侧已增加“快速查询”，可检索：

- 收费项目、价格、目录类资产
- 国家两库规则 item
- 政策依据 chunk
- 原始资料台账

任务结果下方已增加“依据链 / 证据图谱”可视化节点连线图，用于展示规则、收费项目、代码、来源资产和疑点结果之间的关系。
